## Firebase authentification using Reactjs

### Firebase Account
before you start create an account on firebase website and update the file firebaseConfig.js under /src folder
 ```
    apiKey: "******",
    authDomain: "******",
    databaseURL: "******",
    projectId: "******",
    storageBucket: "",
    messagingSenderId: "******",
    appId: "******"
 ```
### Satrt project
 ```
npm install
 ```
npm start
 ```
